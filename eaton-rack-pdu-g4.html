Model Tiers

Four Configurations to Match Every Need

Explorer Standard G4 Models are available in four power management tiers, from straightforward power delivery to full outlet-level monitoring and control.

⚡

Basic G4

Cost-effective, reliable power distribution with a secure built-in high-retention system to firmly hold standard power cables. No network interface required.

  • Reliable power delivery
  • High-retention outlet system
  • P-lock compatible C39 outlets
  • LSZH halogen-free cable
  • Toolless mounting buttons
📊

Metered Input G4

Streamlines power management by simplifying load balancing and preventing overloads through network-level input monitoring and remote visibility.

  • All Basic G4 features
  • Input-level power metering
  • Remote & on-site monitoring
  • Load balancing support
  • Overload prevention
  • Gigabit Network Module (GNM)
🔀

Switched (Metered Outlet) G4

Combines advanced outlet-level control with all power quality measurement capabilities of the Metered Input PDU, enabling remote power cycling per outlet.

  • All Metered Input features
  • Outlet-level monitoring
  • Remote outlet switching
  • Per-outlet power cycling
  • Color-coded outlet sections
  • Cascade up to 40 PDUs
🛡️

Managed G4

The most capable tier. Actively monitors and measures voltage, current, and power consumption at the outlet level with full cybersecurity certification.

  • All Switched G4 features
  • Outlet-level V, A, W measurement
  • Zero-trust architecture
  • Zero-touch provisioning
  • Hot-swappable GNM
  • Dual GbE ports
  • Individual IP per cascaded PDU

Key Features

What Sets the G4 Apart

Every Explorer Standard G4 Model is built around a set of innovations that simplify deployment, improve reliability, and future-proof your data center investment.

🔌

C39 Universal Outlet

The revolutionary single C39 outlet accepts both C13 (10A) and C19 (16A) plugs. P-lock compatible for a firm, secure connection — no adapter needed.

🔁

Hot-Swappable Network Module

The Gigabit Network Module (GNM) can be removed or upgraded without interrupting power to the PDU, eliminating maintenance windows for firmware upgrades.

🔗

Network Cascading

Dual GbE ports let you cascade up to 40 PDUs sharing a single network connection, each with its own individual IP address, dramatically reducing IP consumption.

🎨

Color-Coded Outlet Sections

Visually distinct outlet groups make cable management and phase identification fast and intuitive, reducing human error during installation and maintenance.

🔧

Toolless Mounting

Toolless mounting buttons provide multiple mounting options to fit your rack layout and power needs without requiring extra hardware or specialized tools.

🌿

LSZH Cable (Halogen-Free)

Low Smoke Zero Halogen (LSZH) cables come as standard, reducing toxic gas emission in the event of fire — critical for dense, enclosed data center environments.

🌡️

High-Temp Operation

Rated for high-temperature environments to support modern, energy-efficient data center designs that operate at elevated inlet temperatures.

📡

Zero-Touch Provisioning

Managed models support zero-touch provisioning via the integrated GNM, enabling rapid, automated deployment across large-scale data center rollouts.
